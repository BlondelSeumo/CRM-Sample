Dashboard

View an overview of your application here. You can,

  • View a chart for an overview of your Deals.
  • View a pie chart of your Open, Won and Lost Deals.
  • View total number of your Leads.
  • View total number of your Employees.
  • View total number of Sent and Accepted Proposals.
  • View Rate of Sending and Acceptance of the Proposals.
  • View how many Pipelines do you have.
  • View how many Deals are on your individual Pipeline.
  • And view your top 5 Owners.

Leads

Here you can see and manage all your business leads. A Lead can be a People or an Organization with whom you make deals for your business. You can create a Lead Type and Add People or Organization under that Lead Type.

People

Here you can see and manage all your People type Leads. People are one type of Lead of your business. This is a single person with whom you may have business relationships.

Deals

The business contract or communication that you have with one or more of your Leads is a Deal. In this module the system allows you to manage All your Deals. You can Add a Deal, Add a Pipeline, view and manage the Deals in the Pipelines and Add a reason if a Deal is lost.

Deals View

Here you can see and manage all your Deals in a Table with Status, Deal Value, Organization the deal with, People the deal with Owner of the Deal and Tags. The table data can be filtered with different Filters like Owner, Type, Status, Tags etc.

Deals Card View

Here you can see and manage all your Deals in card view with Status, Deal Value, Organization the deal with, People the deal with Owner of the Deal and Tags. The card data can be filtered with different Filters like Owner, Type, Status, Tags etc.

Deals Discussion Board

Here you can see and manage your Deal in deal discussion modal, Deal Value, Lead and Contact person of the deal, the deal with Owner of the Deal and Tags etc. You can also comment here.

Pipeline

It is a responsive Kanban view where you can manage your Deals in a better visualized way.

Functions that you can perform here,

  • Add a Pipeline with a different no of stages and stage name.
  • In the Pipeline view, all deals are visible and manage the stage of the Deals very easily, Add an Activity directly to the Deal, Edit, Mark and Manage its Tag

Proposals

Here you can manage all your business Proposals like Create a template and Send Proposals. A Business Proposal is a written formal document to a prospective lead

Proposal List

Manage all your Proposals from here. The data table will show you the Status, if Accepted, of which Deal, etc. Filter the table data

Functions that you can perform here,

  • Create and Send a Proposal directly
  • Check the status of your proposals
  • Accept them with a simple toggle button
Templates

Create Templates which you can use to send Proposals and view them and edit them if required from here.

Functions that you can perform here,

  • Add and Edit any Template
  • Duplicate them per requirement
  • Add the Template directly to a Proposal
  • Download the template as HTML for using elsewhere
Proposals Kanban view

Here you can see and manage all your Proposals in kanban view with Status. The kanban view data can be filtered with different Filters like Owner, Tags, Created date etc.

Activities

All your Activities related to a Deal or a Lead you can view and manage from here. An Activity can be of different types, Call, Meeting, Email, Task, etc.

Calendar view

View all your Activities in a Calendar view. Add an Activity, Filter them with different filters, easily search your activities. Change the calendar view by Month, Week or Day.

Functions that you can perform here,

  • Add an Activity with the “Add Activity” button directly.
  • Add an Activity from the calendar to a specific day or time.
  • Simple edit by clicking on the Activity from the calendar view
List view

View all you Activities in a list view under a table view from you. Edit or Delete the Activities from the table. Filter the data of the table with different filters like Schedule, Owner, if the Activity is with a Deal, Person or an Organization.

Functions that you can perform here,

  • Add, Edit or Delete an Activity
  • Mark them if they are Done

Reports

You can view a few types of reports.

Deal

View Deal Owners performance report here.

Functions that you can perform here,

  • Filter data according to Won or Lost.
  • Filter data according to your Pipelines.
  • View Several information from data table
Proposal

View Proposal Owners performance report here.

Functions that you can perform here,

  • Filter data according to Pipelines.
  • Filter data according to Send Proposal and Accepted Proposal.
  • View Several information from data table
Pipeline

View data based on how many deals are in the pipelines according to Owners.

Functions that you can perform here,

  • Filter data according to Pipelines.
  • Filter data according to New Deals, On Processing Deals and Closed Deals.
  • View Several information from data table

User & Roles

Create application Users and assign them particular Roles from here.

Functions that you can perform here,

  • Send invitations to email addresses to add them as application users.
  • Filter them as All Users, Active, Inactive and Invited.

Settings

Set your various settings related to the application.

Functions that you can perform here,

  • Manage application General settings
  • Add required Custom Fields for Leads and Deals
  • Add Email service for User invitation, Sending Proposals and Receiving Notifications
  • Manage Notifications from Notification settings
  • Check for newer updates and Update for New Features and Bug fixes

Importing Guidelines

  • Format your CSV the same way as the sample file
  • Your CSV columns should be separated by commas, not semicolons or any other characters.
  • The number of columns in your CSV should be the same as the sample.
  • Recommendation for importing huge data, you have to increase your server execution time and memory limit.
  • Required field's column cell must not be empty.
  • Lead column's cell format should be like, P: Valid_Person_Name or, O: Valid_Organization_name: Valid_Contact_Person_Name, Here O and P is for the Lead type which indicates Organization and Person respectively.
  • Make sure you are importing/adding data in this sequence Lead Type, Person/Organization, Pipeline, and then Deals.
  • When an error occurs download the error file and correct the incorrect cells and import that file again through .csv format.
